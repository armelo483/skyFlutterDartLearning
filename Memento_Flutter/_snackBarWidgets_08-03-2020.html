Composition de l'application:

 

Ce code est constitué des widgets suivants:

MaterialApp(0): Le Widget de base de type Material Design qui englobe les widgets flutter de l'appli
et rajoute des widgets spécifiques de Material Design comme "Scaffold"

Scaffold: Widget de base définit pour la page
AppBar: Gère la navBar
Text: Pour afficher le texte dans de la navBar

SnackBarPage: classe pour utiliser le widget SnackBar qui utilise  "Center" et ayant pour enfant "RaisedButton"
 Utilisé pour plus de visibilité dans le code

Center: Pour centrer 
RaisedButton: Widget material permetant facilement de parametrer un boutton avec 
une fonction repondant aux éveenments de clic. Ce qui aurait été plus difficile
sans passer par material
